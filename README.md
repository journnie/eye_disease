- **개요**
    
    본 프로젝트는 반려견의 건강 관리와 피부 질환에 대한 조기 진단 및 관리를 돕기 위한 반려견 피부 질환 진단 예측 시스템을 개발하는 것을 목표로 한다.

- **주요 기능 및 서비스**
    - 피부 질환 진단 예측
